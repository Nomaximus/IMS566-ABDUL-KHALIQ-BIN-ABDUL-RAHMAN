# Student Registration System

A responsive web application designed for managing student enrollment records, combining a clean Bootstrap user interface with back-end logic.

## ✨ Features Included
* **Responsive Layout:** Embedded inside a mobile-friendly fixed-width container and structured row grid system.
* **Desktop Grid Adaptation:** Dynamically scales to exactly half-screen width (`col-md-6`) on desktop/laptop views and stacks beautifully on mobile displays.
* **Data Capture Form:** Specialized validation inputs configured to collect Name, Email, and Age safely.
* **Dual Action Routing:** Outfitted with a primary "Save" submission button and a danger-themed "Clear" reset button.
* **Secure Access Gateway:** Protects the registration page using a mock/integrated login verification mechanism.
